IBSU ბექენდის პროგრამირების პროექტი

ავტორი: ირაკლი ლაფაჩი

პროექტის აღწერა:
პროექტი მოიცავს სანერგის ონლაინ ვაჭრობის(E-Commerce) პლატფორმის
ძირითად ფუნქციონალს. აპლიკაციის დასტარტვისას ინიცირდება სატესტო მონაცემები.
პროექტში გაერთიანებულია მომხმარებლების CRUD და 
ავტორიზაცია, პროდუქტების და მათი კატეგორიების CRUD, კალათის
და შეკვეთების ისტორიები. პროექტში ასევე მოცემულია API controllerები
ამ ფუნქციონალისათვის.

პროექტი აწყობილია Spring Boot Frameworkზე.
